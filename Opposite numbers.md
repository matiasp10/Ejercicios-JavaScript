Very simple, given a number, find its opposite.

Dado un numero, encontrar el opuesto
## Ejemplos

```
1: -1

14: -14

-34: 34
```

## Mi soluci√≥n

````js
function opposite(number) {
	// Solo devuelvo el inverso del numbero ingresado
	return -number;
}
````
